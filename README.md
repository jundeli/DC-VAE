# DC-VAE
Official Code for DC-VAE paper (CVPR 2020): https://arxiv.org/abs/2011.10063. 

## Setup conda environment
 - `conda env create -f dcvae_env.yml`
 - `conda activate dcvae_env`

## CIFAR-10 training code
 - `python train.py`
